# Todo-Model

A simple and efficient Todo application built with JavaScript and HTML.

## Overview

This project provides a basic Todo list model for managing tasks. It demonstrates fundamental concepts of web development using vanilla JavaScript and HTML.

## Tech Stack

- **JavaScript** (69.3%) - Core application logic
- **HTML** (30.7%) - Markup and structure

## Features

- Create new todo items
- View all todos
- Manage your task list

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Shadan0786/Todo-Model.git
cd Todo-Model
```

2. Open `index.html` in your web browser to run the application.

## Usage

Open the application in your browser and start managing your todos. The interface allows you to:
- Add new tasks
- Track your progress
- Organize your daily work
